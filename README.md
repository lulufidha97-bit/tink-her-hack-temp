### Project Description
Cloth Matching & Outfit Tracker is a smart fashion assistant web application that helps users mix and categorize their wardrobe, and get outfit suggestions based on color combinations and style preferences. It solves the daily dilemma of "what to wear" by digitizing your closet.
### The Problem statement
Many people struggle to decide what to wear daily. They forget what clothes they own, find it difficult to match outfits properly, and often end up repeating the same combinations.
### The Solution
This app allows users to:
- Upload and store their clothes digitally.
- Get smart, randomized outfit suggestions.
- Save favorite combinations for easy access.
- Track previously worn outfits.

## Features
- **Digital Wardrobe:** Add and categorize clothes (Tops, Bottoms, Shoes) with images.
- **Outfit Generator:** Randomly generates a matching Top and Bottom combination.
- **Favorites System:** Save your favorite generated outfits to a dedicated list.
- **Outfit History:** conceptual tracking of daily suggestions.
- **Responsive Design:** Works seamlessly on mobile and desktop devices.
- **Data Persistence:** Your closet and preferences are saved locally in your browser.

## Implementation
### Installation
1. Clone the repository:
```bash
git clone https://github.com/lulufidha97-bit/tink-her-hack-temp.git
```
2. Navigate to the project directory:
```bash
cd tink-her-hack-temp
```
### Run
Simply open the `index.html` file in your web browser.
Or use a local server (optional):
```bash
# using python
python -m http.server
# or using live-server
live-server .
```
